# Mobile QA (v1.0)

Run on a **phone** (and tablet if you care) over **cellular** or guest Wi‑Fi, using the **same public URL** you ship. Note device, OS, browser, and pass/fail.

**Login:** No sideways scroll; Sign In stays reachable with keyboard. Login works; refresh still logged in.

**Dashboard:** Portrait + landscape OK; tiles open a camera; live video within ~15s; menu opens/closes (backdrop closes too).

**Camera page:** Live plays; changing mode doesn’t break the page; back returns to dashboard.

**Recordings:** Picker works narrow; playback starts; timeline works with a **finger** (not hover-only).

**Events** (if you use them): List reachable; one clip plays.

**Settings** (admin): Configuration → Settings scrolls and saves.

**PWA (optional):** Add to Home Screen / Install still opens the same site and logs in.
